# eslint-config-rakko

ESLint 规则集

## 安装

```bash
pnpm add -D eslint-config-rakko
```

在 `.eslintrc.js` 中添加：

```javascript
module.exports = {
  extends: ['rakko', 'rakko/react']
}
```
